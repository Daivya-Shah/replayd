replayd

Reproduce any agent failure in one click.

A transparent proxy that records every step of your LLM agent's runs, so you can replay, branch, and regression-test them — deterministically, offline.

Quickstart View on GitHub

Open source · self-hosted · your API key is never stored.

The problem

Production failures you cannot replay are failures you cannot fix.

LLM agents fail nondeterministically in production. Request logs show fragments, not the full trajectory. Without a byte-faithful recording of each step, you cannot reliably reproduce the exact run, inspect what the model returned, or prove a fix does not regress.

How it works

Three steps from capture to debug.

  1. 01

    Point your agent's OpenAI-compatible client at replayd. Every request and response is captured transparently — your agent behaves as if it were talking to the provider directly.

    base_url="http://localhost:8787/v1"
  2. 02

    Open the dashboard to browse runs step by step. Inspect request bodies, response bodies, latency, and model metadata for any exchange in the recording.

  3. 03

    Replay a run deterministically from storage with no upstream API calls, or branch from a parent run to change one step and execute the rest live against the model.

Capabilities

Built for agent reliability work.

Deterministic replay

Re-run agent logic against recorded responses. Same inputs produce the same outputs — no network, no drift.

Branch and test

Change a prompt, model, or tool at one step; replay matching steps from the parent and go live on divergence.

Semantic regression tests

Pin a baseline run and compare fresh recordings. Tolerates wording changes; catches real behavioral drift in tool calls and finish reasons.

Trust

Your infrastructure, your data.

  • Bring your own key. The agent sends its provider API key on each request. replayd forwards it upstream, redacts it from stored headers, and never persists or bills for model usage.
  • Open source and self-hostable. Proxy, control plane, and dashboard run entirely in your environment. One command with Docker Compose; no external dependencies beyond your LLM provider.

Quickstart

Running locally in under a minute.

git clone <repo>
cd replayd
docker compose up --build

Dashboard at http://localhost:3000 · point your agent at http://localhost:8787/v1
